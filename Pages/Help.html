Folders

The Content Folder is for assets like music, sound, data files, images, and whatever else needed.

The SP Folder is for the script files, which I call script packets for this project.

The Pages Folder is for different html files and the .css files for stylization which are in the Styles folder inside the Pages folder.

Contributing

As long as you have Javascript, HTML, A dream and CSS experience. Even the bare minimum of knowledge of how they work. You can Contribute.

Just make a fork, code your changes, we review and if you have all the checkmarks, you have just added to the site and made your way into the credits!
